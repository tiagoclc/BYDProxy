Proxy para compartilhar a internet via hotspot do BYD Dolphin (e outros com dilink 3).

Basta instalar o apk na multimídia e liberar a inicialização automática.
Quando conectar o celular ou outro dispositivo ao hotspot, deve ir nas configurações da rede, no aparelho que conectou, para inserir o proxy.
Deve escolher a opção de proxy automático e definir o seguiinte endereço:
http://192.168.43.1/proxy.pac (o ip deve ser o mesmo do hotspot 192.168.43.1 é o ip padrão do hotspot do byd dolphin. Talvez seu carro tenha outro endereço).
Com isso o hotspot do carro deve compartilhar a internet para os seus dispositivos!
